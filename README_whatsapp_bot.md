# WhatsApp LangChain Bot

This repository contains a WhatsApp chatbot built with **LangChain** and **OpenAI**, featuring:

- Custom tools: Order Lookup, Product Finder, FAQ
- Optional RAG (Retrieval-Augmented Generation) using FAISS vectorstore
- Google Sheets analytics logging
- Dockerized with Prometheus metrics for monitoring
